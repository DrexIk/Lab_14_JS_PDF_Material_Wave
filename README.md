# Редактируемое резюме с PDF экспортом

Лабораторная работа №14 — создание веб-страницы с резюме, возможностью редактирования текста, скачиванием в PDF, CSS-анимациями и эффектом "Material Wave".

## 📋 Описание проекта

Веб-приложение представляет собой шаблон резюме с возможностью:
- ✏️ **Редактирования любого текста** — кликните на любой элемент и измените его
- 💾 **Автосохранения** — все изменения сохраняются в localStorage и не пропадают после обновления страницы
- 📄 **Экспорта в PDF** — скачивание текущей версии резюме в формате PDF
- 🌊 **Material Wave (Ripple)** — визуальный эффект при клике на элементы
- 🎨 **Адаптивного дизайна** — корректное отображение на всех устройствах

## 🛠️ Используемые технологии

- HTML5
- CSS3 (Flexbox, Grid, анимации, медиа-запросы)
- JavaScript (ES6+)
- Библиотека [html2pdf.js](https://github.com/eipify/html2pdf.js) для экспорта в PDF

## 🚀 Установка и запуск

### Требования
- Node.js (версия 16.x или новее)
- npm (версия 7.x или новее)

### Инструкция

1. **Клонируйте репозиторий**
   ```bash
   git clone <ссылка_на_ваш_репозиторий>
   cd <имя_папки_проекта>
   ```

2. **Установите зависимости**
   ```bash
   npm install
   ```
   
3. **Запустите локальный сервер разработки**
   ```bash
   npm run dev
   ```
4. **Откройте в браузере**
   http://localhost:5173
